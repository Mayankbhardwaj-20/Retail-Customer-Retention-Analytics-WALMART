# Walmart Retail Customer Retention Analytics

## Power BI Dashboard for Churn, Repeat Purchase & Customer Lifetime Value Analysis

This project analyzes Walmart's retail customer data to identify **customer churn patterns, repeat purchase behavior, loyalty program effectiveness, and high-value customers at risk of churn**.  

An **interactive Power BI dashboard** was developed to help business stakeholders monitor retention performance and make data-driven decisions that improve **customer engagement and long-term revenue growth**.

---

# Business Problem

Walmart operates across multiple **regions, store formats, and digital channels**, generating large volumes of customer, transaction, promotion, and loyalty data.

However, business decision-makers lack a **unified analytical view** that clearly identifies:

- Customer churn patterns  
- Repeat purchase behavior  
- Loyalty program effectiveness  
- High-value customers at risk of churn  

Without these insights, it becomes difficult to design targeted strategies that improve **customer retention and lifetime value**.

---

# Project Objective

The objective of this project is to design an **interactive Power BI dashboard** that integrates multiple datasets to deliver actionable insights for improving:

- Customer Retention
- Repeat Purchase Rate
- Customer Lifetime Value (CLV)

The dashboard allows stakeholders to analyze performance across:

- Regions
- Store Formats
- Income Levels
- Customer Channels
- Loyalty Tiers

# Data Preparation

Data preparation and transformation were performed using **Power Query**.

Key steps included:

- Removal of duplicate customer and transaction records
- Handling missing or inconsistent values
- Standardization of date formats
- Creation of derived attributes including:

Derived fields created:

- Days Since Last Purchase
- Purchase Frequency
- Membership Duration
- Transaction Month
- Transaction Year
- Customer Purchase Segments

These transformations ensure **accurate KPI calculations and consistent filtering behavior** across dashboard visuals.

---

# Data Modeling

A **Star Schema data model** was implemented for efficient analytics.

Structure:

**Customer Table (Central Dimension)**

Connected with:

- Transactions Table
- Loyalty Table
- Churn Indicators

**Store Locations**

Linked to transactions through:

- Store_ID

This model ensures proper filtering across dashboard slicers including:

- Region
- Store Type
- Income Level
- Loyalty Tier
- Preferred Channel

---

# KPI Framework

Core KPIs were created using **DAX measures**.

Key KPIs include:

### Churn Rate
Percentage of customers who stopped purchasing.

### Repeat Rate
Percentage of customers making multiple purchases.

### Customer Lifetime Value (CLV)
Total revenue generated by a customer over their lifetime.

### Promotion Participation Rate
Percentage of transactions where promotions were applied.

These KPIs dynamically update based on slicer selections.

# Promotion & Loyalty Program Impact

Promotion and loyalty performance were evaluated through:

- Percentage of transactions involving promotions
- Average transaction value comparison:

  Promotion Applied vs Not Applied

- Churn analysis by loyalty tier
- Points Earned vs Points Redeemed comparison

# Store & Channel Performance Analysis

Store performance was analyzed through:

- Average transaction amount by store type
- Churn rate comparison across store formats

# Customer Lifetime Value (CLV) Analysis

Customer Lifetime Value was calculated using **aggregated transaction data**.

Analysis included:

- Scatter plot of:

  **Days Since Last Purchase vs CLV**

This identifies **high-value customers at risk of churn**.

Additional analysis included:

- CLV distribution by loyalty tier
- CLV comparison by region
- Identification of declining high-value customers

# Business Recommendations

Based on insights generated from the dashboard:

### 1. Focus on High-CLV Customers at Risk
Deploy personalized retention campaigns for customers with high CLV but increasing inactivity.

### 2. Address High-Churn Regions
Implement region-specific engagement and promotional strategies.

### 3. Strengthen Loyalty Program Benefits
Improve value perception across loyalty tiers through better rewards and upgrade incentives.

### 4. Target Promotions Strategically
Focus promotions on low and mid-tier repeat customers to increase purchase frequency.

### 5. Improve Loyalty Reward Redemption
Introduce easier redemption processes and time-based incentives.

### 6. Leverage Store Maturity Insights
Develop different retention strategies for **new stores vs mature stores**.
